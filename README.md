# Find MokUp

내가 작업했던 디자인 시안을 올려두어 년도별, 카테고리별로 나중에 쉽게 찾자!

![리스트페이지 스크린샷](./screenshot/list.jpg)
![등록 스크린샷](./screenshot/register.jpg)

## 주요 기능

- [ ✓ ] 로그인, 로그아웃
- [ ✓ ] 년도별 필터 정렬
- [ ✓ ] 카테고리별 필터 정렬
- [ ✓ ] 목업 등록 및 수정 삭제

## Stack

- React
- Redux
- Redux Saga
- Typescript
- Firebase
